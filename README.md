# Página web personal — Guillermo Droppelmann, PhD

Sitio: https://guillermodroppelmann.github.io

## Estructura

- `index.html` — About: perfil, áreas de investigación, cargos, formación, proyectos, premios, comités (incluye datos schema.org para Google)
- `publications.html` — Articles: 43 artículos espejo de Google Scholar, con botón **Cite** y link a **Google Scholar**
- `books.html` — Books: libro editado (Amazon) + 13 capítulos con primera plana visualizable y botón **Cite**
- `talks.html` — Upcoming talks + 40 presentaciones agrupadas por año
- `teaching.html` — docencia, supervisión de tesis de magíster y mentoría
- `images/` — foto de perfil y primeras planas de capítulos

## Mantenimiento

- **Nuevo artículo:** editar `publications.html` copiando un bloque `<li class="pub">...</li>`
- **Nueva charla:** editar la sección Upcoming en `talks.html`
- **Nueva primera plana:** subir la imagen a `images/chapters/` y referenciarla en `books.html`

## Visibilidad

1. **LinkedIn:** agrega la URL del sitio en Información de contacto → Sitio web
2. **Google Scholar:** agrega la URL en tu perfil → Página principal
3. **Google Search Console:** registra el sitio y envía `sitemap.xml`
